Overview

This system works by processing an array of actions one at a time. Each action must complete before the next action begins. 

Move Susie
↓
Wait for arrival
↓
Display text
↓

↓

↓

Action Reference

action_move()

action_move(obj, x, y, speed)

Moves any object directly to a position.

action_move_actor_direct()

action_move_actor_direct("Susie", x, y, speed)

Moves an actor diagonally toward a target.

action_move_actor_nondirect()

action_move_actor_nondirect(
    "Susie",
    x,
    y,
    speed,
    PREFER.VERT
);

Moves vertically then horizontally, or vice versa.

action_wait_time()

action_wait_time(2)

Waits 2 seconds.

action_wait_for_object()

action_wait_for_object(obj_boss)

Pauses until an object exists.

action_wait_for_object_destroy()

action_wait_for_object_destroy(obj_textbox)

Pauses until an object is destroyed.

action_display_text()

action_display_text(
    ["Hello World"],
    snd_text_susie
);

Example Cutscene

function cutscene(){

    with(obj_cutscene_master){

        var actions = [

            action_wait_time(1),

            action_move_actor_nondirect(
                "Susie",
                obj_checkpoint_a.x,
                obj_checkpoint_a.y,
                2,
                PREFER.VERT
            ),

            action_display_text(
                ["You notice that susie has moved. How cool!"]
            ),

            action_wait_for_object_destroy(obj_textbox),

            action_actor_change_sprite(
                "Susie",
                spr_susie_laugh
            ),

            action_play_sound(
                snd_susie_laugh,
                true
            )

        ];

        start_cutscene(actions);

    }

}

Best Practices

  1. Always wait for textboxes to close.
  2. Give every actor a unique actor_id.
  3. Use checkpoints instead of hardcoded coordinates.
  4. Keep room transitions at the end.
  5. Use nondirect movement for Deltarune/Undertale like feel scenes.
  6. Only have one obj_cutscene_master per room.
  7. Keep cutscene logic inside dedicated scripts.
